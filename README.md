# travelX (TRVLX) - Sri Lanka Premium Tourism Platform

A high-end, minimalist tourism landing platform showcasing iconic destinations in Sri Lanka. Built with pure HTML, vanilla CSS, and JavaScript.

## Features
- Minimalist editorial layout with sharp visual geometry.
- Strict WCAG 2.2 AAA accessibility standard adherence.
- Asymmetric transition grids and custom off-canvas itinerary drawer.
- Stone/Charcoal (Dark) and Cream/Slate (Light) presets.
